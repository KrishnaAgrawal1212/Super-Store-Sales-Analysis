# Super-Store-Sales-Analysis

Project Overview

This project transforms raw retail sales data into an interactive Power BI dashboard that analyzes sales, profit, customer segments, payment methods, shipping modes, and regional performance. The dashboard helps businesses make data-driven decisions through clear visualizations and actionable insights.

Learning

Applied data analysis techniques, including time series analysis, to generate meaningful insights, support sales forecasting, and create interactive dashboards for business growth.

Key Features

1.Data Modeling: Structured and optimized sales data for efficient reporting and analysis.

2.DAX Calculations: Created custom measures for KPIs such as Sales, Profit, Quantity, and Delivery Time.

3.Time Series Analysis: Analyzed monthly sales and profit trends to identify business patterns.

4.Interactive Visuals: Developed dynamic charts and graphs for category, segment, and regional analysis.

5.KPI Tracking: Monitored key business metrics through intuitive KPI cards.

6.Cross-Filtering: Enabled interactive filtering across visuals for deeper data exploration.

7.Business Insights: Delivered actionable insights to support data-driven decision-making.

Key Insights

1.Total Sales reached 1.57M with 175.26K profit.

2.Consumer Segment contributed the highest sales (48%).

3.Office Supplies was the top-performing category (0.64M).

4.Phones generated the highest sales among sub-categories (0.20M).

5.COD was the most preferred payment method (43%).

6.West Region led overall sales performance (33%).

7.Sales and profit peaked during the final quarter of the year.

Project Conclusion

This project demonstrates the development of a comprehensive and interactive Sales Analytics Dashboard using Power BI. Through advanced data modeling, DAX calculations, time series analysis, and compelling visualizations, the dashboard delivers actionable insights into sales performance, profitability, customer behavior, and regional trends. These insights empower businesses to optimize operations, improve customer targeting, and drive sustainable growth.
